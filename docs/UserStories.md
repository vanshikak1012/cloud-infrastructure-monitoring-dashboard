# User Stories

## Story 1

As an IT Operations Manager,
I want to monitor infrastructure health,
so that I can quickly identify performance issues.

---

## Story 2

As a Business Manager,
I want to view operational KPIs,
so that I can make informed business decisions.

---

## Story 3

As a System Administrator,
I want to monitor Docker containers,
so that I can detect resource bottlenecks.

---

## Story 4

As a Cloud Engineer,
I want centralized monitoring,
so that I can troubleshoot efficiently.
