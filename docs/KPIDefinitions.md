# KPI Definitions

| KPI | Description | Business Value |
|-----|-------------|----------------|
| CPU Utilization | Percentage of CPU usage | Indicates server workload |
| Memory Utilization | RAM consumption | Helps identify memory bottlenecks |
| Disk Usage | Storage utilization | Prevents storage issues |
| Network Traffic | Incoming/Outgoing traffic | Monitors connectivity |
| System Availability | System uptime | Measures service reliability |
| Active Targets | Monitored services | Ensures monitoring coverage |
